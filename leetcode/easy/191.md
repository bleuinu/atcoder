# 191. Number of 1 Bits
a.k.a. [Hamming weight](https://en.wikipedia.org/wiki/Hamming_weight)

```cpp
class Solution {
public:
    int hammingWeight(uint32_t n) {
        int cnt = 0;
        for(int i=0; i<32; ++i) {
            if(n&1) ++cnt;
            n >>= 1;
        }
        
        return cnt;
    }
};
```
